# 多语种在线学习平台 — 产品需求文档

## 1. 产品概述

LinguaVerse 是一款沉浸式多语种在线学习平台，支持英语、日语、韩语等主流语言，通过分级课程体系、互动式学习模块和个性化推荐，为语言学习者打造高效且有趣的学习体验。

- 目标用户：有语言学习需求的各年龄段用户，从零基础到高级水平
- 核心价值：将游戏化、社区化与科学学习方法结合，让语言学习不再枯燥

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 学习者 | 邮箱注册 | 浏览课程、学习模块、追踪进度、社区互动 |
| 管理员 | 后台指定 | 课程管理、用户管理、内容审核 |

### 2.2 功能模块

1. **首页**：品牌展示、语言选择入口、学习数据概览、热门课程推荐
2. **课程中心**：分级课程浏览、课程详情、课程筛选
3. **学习模块**：单词记忆、语法练习、口语跟读、听力训练四大互动模块
4. **学习进度**：学习数据仪表盘、每日打卡、学习日历、能力雷达图
5. **个人中心**：注册登录、个人信息管理、学习路径推荐、成就展示
6. **社区**：学习动态、讨论区、学习伙伴匹配
7. **成就系统**：徽章收集、等级晋升、学习排行榜

### 3.2 课程学习流程

浏览课程中心 → 筛选目标课程 → 查看课程详情 → 按章节顺序学习 → 完成章节测验 → 解锁下一章节 → 完成课程获得证书

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝 (#0F172A) + 活力橙 (#FF6B35) 作为强调色，搭配暖白 (#FAFAF9) 背景
- **辅助色**：薄荷绿 (#10B981) 表示成功/完成，珊瑚红 (#EF4444) 表示错误/警告
- **按钮风格**：圆角胶囊按钮，主按钮使用渐变效果，悬停时有微动效
- **字体**：标题使用 Outfit（现代几何感），正文使用 Noto Sans SC（中文友好）
- **布局风格**：卡片式布局，左侧导航栏 + 主内容区，大量留白
- **图标风格**：Lucide 线性图标，2px 描边
- **整体氛围**：温暖、友好、充满活力，融合游戏化元素

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 平板适配（768px-1279px）：侧边栏折叠为顶部导航，卡片网格从4列变2列
- 移动端适配（<768px）：底部Tab导航，单列卡片，学习模块全屏沉浸模式
- 触控优化：按钮最小点击区域 44x44px，滑动手势支持

### 4.4 动效设计

- 页面切换：淡入淡出 + 轻微上移
- 卡片悬停：阴影加深 + 轻微上浮
- 答题反馈：正确时绿色脉冲 + 粒子爆发，错误时红色摇晃
- 进度更新：数字滚动动画 + 进度条平滑填充
- 成就解锁：徽章从中心放大弹出 + 金色光晕
